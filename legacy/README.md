# Legacy Archive

This folder preserves the pre-pivot RL-first codebase and experiment assets.

Archived scope:

- `legacy/packages/ood_solver/`: structured RL model and environments
- `legacy/scripts/`: RL training/evaluation utilities
- `legacy/configs/`: bridge/minigrid RL experiment configs
- `legacy/docs/`: RL-specific protocols and plans
- `legacy/requirements-benchmarks.txt`: RL benchmark deps

Policy:

- No new feature development in `legacy/`.
- Historical reruns are allowed for reproducibility.
- Active research and code changes happen in the top-level LLM-agent track.
